# Data

No raw data are included in this public package.

The paper used Minnesota crash records, MnDOT ATR hourly traffic exposure,
weather, gasoline-price, and sunrise/sunset/daylight controls. Raw crash mirrors,
Access databases, and large SAS data tables are intentionally omitted. The
repository preserves paper-facing SAS code and public/source pointers rather than
redistributing a local agency-data mirror.
