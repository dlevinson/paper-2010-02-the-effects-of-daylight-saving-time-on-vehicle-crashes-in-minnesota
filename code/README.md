# SAS Code

This folder contains the three paper-facing SAS programs preserved for the
Daylight Saving Time crash analysis package.

- `Arthur.sas`: crash-period summaries and negative-binomial models.
- `SunriseTime.sas`: sunrise/sunset and DST transition variables.
- `mncrashanalysisJune30.sas`: traffic-exposure preparation, data merges, and main crash models.

The scripts expect SAS-era source tables that are not included here. See
`../documentation/PUBLIC_SOURCE_LINKS.csv` and
`../documentation/NONCOPIED_SOURCE_MANIFEST.csv` for provenance and exclusions.
